# 📄 Currículo

Pasta com o meu currículo profissional em PDF e o script Python que o gera.

## 📁 Arquivos
* **`Renan_Silva_dos_Santos_CV.pdf`** — versão final do currículo, pronta para download.
* **`gerar_cv.py`** — script Python que monta o PDF a partir das informações do perfil (formação, experiência, skills e idiomas).

## 🔧 Como gerar novamente o PDF
1. Instale a dependência: `pip install reportlab`
2. Rode o script: `python gerar_cv.py`
3. O arquivo `Renan_Silva_dos_Santos_CV.pdf` é gerado na mesma pasta.
